## Sobre o Projeto

Este projeto tem como objetivo realizar uma análise estatística básica utilizando dados simulados do agronegócio, com foco na interpretação de variáveis quantitativas e qualitativas.

A atividade foi desenvolvida utilizando a linguagem R, aplicando conceitos fundamentais de análise exploratória de dados.

---

## Estrutura da Base de Dados

A base foi construída manualmente no formato CSV e contém 30 registros com as seguintes variáveis:

- Colheitas → variável quantitativa discreta  
- Producao_kg_ha → variável quantitativa contínua  
- Cultura → variável qualitativa nominal  
- Irrigacao → variável qualitativa ordinal  

---

## Análise Realizada

A variável escolhida para análise foi Producao_kg_ha, onde foram aplicadas:

### Medidas de Tendência Central
- Média  
- Mediana  
- Moda  

### Medidas de Dispersão
- Variância  
- Desvio padrão  
- Amplitude  

### Medidas Separatrizes
- Quartis  
- Quintis  
- Decis  

---

## Análise Gráfica

Foram gerados os seguintes gráficos:

- Histograma da produção por hectare  
- Boxplot da produção  
- Gráfico de barras da variável Cultura  

---

## Observações

A distribuição dos dados indica uma maior concentração na cultura do café, demonstrando maior frequência na base analisada.

Os valores de produção apresentam variação consistente, permitindo análise adequada por meio das medidas estatísticas e visualizações.
